Linking to relative page

Sites link to internal web pages too

HTML files are usually stored in root directory, or main folder

HTML files are often stored in the same folder. If the browser is currently displaying index.html, it can also know about about.html and contact.html if they in the same folder

If they in the same folder, you can link web pages using relative path using the href as the name and value as ./nameOfFile.html

  • A relative path is a filename that shows the path to a local file. The ./ tells the browser to look for the file in the current folder
  • An absolute path is the complete route to a file or directory from the system root.

Linking to same page

Let's make it easier for a user to jump to different portions of the same page

  1. in order to link to a target on teh same page, we must give the target an id attribute
  2. Assign the element (the target ) with the id attribute
  3. ids should be descriptive
  4. href the element that is going to link to that id. it shoulder prepend with a #
  5. thus if we have a p tag with the id, value as top, the element with the href should be #top
