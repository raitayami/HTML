HTML Tables

Introduction to Tables

HTML tables presents information in a two-dimensional table to the users

  1. To create a table, you use table tag
  2. Then you insert a table row element, tr, this creates a row
  3. Then you insert a table data element, td, this cretes a cell
  4. For example. having a table tag, 1 tr, 5 td, will get you a table with one row, two column
  5. Example below, shows this
Data 1 Data 2 Data 3 Data 4 Data 5

There are titles that describe the data

For example, table heading, th, gives more meaning than a simple td

th adds titles to rows or columns, depending on where you put it

Spanning Column

What if the table contains data that spans multiple columns

Data can span columns using the colspan attribute. The attribute accepts an integer, denoting the number of columns it spans across

Using the colspan attirube on the td element

Monday Tuesday Wednesday
Rest Day Gym

In this table, rest day is spanned two columns. It is not that visible due to absence of CSS

Spanning rows

Data can also span multiple rows using the rowspan attribute

Same as colspan, maybe the data for this spans multiple rows, accepting an integer to denote the number of rows it spans across
